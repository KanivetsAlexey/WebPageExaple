!

<!DOCTYPE>
сообщает какую версию HTML вы используете. Расположен в самой первой строчке кода
<!-- -->
добавляет комментарий в код документа. Текст комментария не отображается на странице. Разрешается внутрь комментария добавлять другие теги, вложенные комментарии (когда один комментарий расположен внутри другого) недопустимы

A

<A>
служит для создания ссылок
<ABR>
используеться для аббревиатуры
<ACRONYM>
идентифицирует акроним
<ADDRESS>
используется при записи адресов
<APPLET>
использует файлы с расширением .class. Служит для создания разнобразных эфектов
<AREA>
определяет фигуру и координаты чуствительных областей
*<ASIDE>
Определяет блок сбоку от контента для размещения рубрик, ссылок на архив, меток и другой информации. Такой блок, как правило, называется «сайдбар» или «боковая панель»
*<AUDIO>
добавляет, воспроизводит и управляет настройками аудиозаписи на веб-странице

B

<B>
жирный шрифт
<BASE>
позволяет авторам явно указать базовый URL документа
<BASEFONT>
предназначен для задания шрифта, размера и цвета текста по умолчанию
<BDO>
с устанавливает направление вывода текста и преимущественно предназначен для использования с языками, где чтение происходит справа налево. Например, к ним относится арабский язык.
<BGSOUND>
создает фоновый звук
<BIG>
выводит более крупный текст
<BLOCKQUOTE>
предназначен для включения в документ длинный цитат, создавая отступы с обеих сторон текста и отделяясь от остального текста пустыми строками
<BODY>
определяет видимую часть документа
<BR>
переход на другую строку
<BUTTON>
создает кнопку с рисунком или с другими встроенными элементами

C

*<CANVAS>
Создает область, в которой при помощи JavaScript можно рисовать разные объекты, выводить изображения, трансформировать их и менять свойства
<CAPTION>
определяет заголовок для таблицы
<CENTER>
выравнивает к центру
<CITE>
используется для цитат
<CODE>
отмечает текст как фрагмент программного кода
<COL>
задает ширину и другие характеристики одной или нескольких колонок таблицы
<COLGROUND>
предназначен для задания ширины и стиля одной или нескольких колонок таблицы

D

*<DATALIST>
Создает список вариантов, которые можно выбирать при наборе в текстовом поле. Изначально этот список скрыт и становится доступным при получении полем фокуса или наборе текста
<DD>
является описанием списка определений
<DFN>
отмечает текст как определений
<DIV>
определяет контейнер для HTML. Данный элемент предназначен для использования в CSS
<DL>
служит для создания списка определения
<DT>
является определением списка определений

E

<EM>
<наклонный текст

F

<FIELDSET>
рисует рамку вокруг текста и других объектов
<FONT>
определяет выводимый шрифт, его цвет и размер
*<FOOTER>
задаёт «подвал» сайта или раздела, в нём может располагается имя автора, дата документа, контактная и правовая информация
<FORM>
создает фреймы
<FRAME>
создает фреймы

<Hz>
заголоков, вместо z цифра
<HEAD>
определяет место, где помещается различная информация не отображаемая в теле документа
<HR>
создает полоску
<HTML>
показывает что это HTML документ

I

<I>
наклонный текст
<IFRAME>
создает фреймы
<IMG>
служит для внедрения графики на странице
<INPUT>
предназначен для создания элементов управления и всегда используется совместно с атрибутом TYPE

<KBD>

L

<LEGEND>

<LI>
показыввает элемент списка
<LINK>
определяет ссылку

M

<MAP>
определяет коллекцыю чуствительных областей в графическом изображении
<MARQUEE>
является ответом на BLINK
*<MENU>
служит для создания ненумированного списка
<META>
используется для дачи данных поисковикам

N

*<NAV>
задает навигацию по сайту
<NORB>
запрещает перевод строки

O

<OBJECT>
внедряет объект в html-страницу
<OL>
служит для создания нумерованного списка
<OPTGROUP>
позволяет групировать элементы
<OPTION>
с помощю него создаются строки списков SELECT

P

<P>
создает новый параграф
<PLAINTEXT>
отображает содержимое контейнера «как есть». Пока тег <plaintext> не закрыт, все теги внутри него отображаются как обычный текст. Несмотря на то, что все браузеры поддерживают этот тег, он не входит в спецификацию HTML и его использование осуждается
<PRE>
обрамляет предварительно отформатированный текст
*<PROGRESS>
для отображения прогресса завершенности задачи. Изменение значения происходит через JavaScript

Q

<Q>
предназначен для включения в документ короткой цитаты дез создания отступов

R

*<RP>
для вывода текста в браузерах, которые не поддерживают тег <ruby>
*<RT>
добавляет аннотацию сверху или снизу от текста, заключенного в контейнер <ruby>. Сама аннотация выводится уменьшенным шрифтом
*<RUBY>
предназначен для добавления небольшой аннотации сверху или снизу от заданного текста

S

<S>
делает текст перечеркнутым
<SAMP>
отмечает тексткак образец
<SCRIPT>
используется для включения небольших програм - скриптов (JavaScript, VBScript)
<select>
создает список или выпадающий список
<SMALL>
выводит более мелкий текст
<SPAN>
определяет контейнер для внутреннего текста. Данный элемент предназначен для использования в CSS
<STRONG>
жирный текст
<SUB>
текст в нижнем регистре
<SUP>
текст в верхнем регистре

<TH>

<TITLE>

<UL>

<WBR>

X

<XMP>
отображает содержимое контейнера «как есть» и шрифтом фиксированной ширины. Пока тег <xmp> не закрыт, все теги внутри него отображаются как обычный текст
