<RT>

Описание
Добавляет аннотацию сверху или снизу от текста, заключенного в контейнер <ruby>. Сама аннотация выводится уменьшенным шрифтом. Такая форма записи преимущественно используется для азиатских языков вроде китайского или японского, но также может применяться для других языков, если требуется написать один текст над другим.
Синтаксис
<ruby>
 текст
 <rt>аннотация</rt>
</ruby></pre>
Атрибуты
Нет.
Закрывающий тег
Не обязателен, если идет последним
Пример
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title<ке>/title>
    <tyle>
   .date { font-size: 1.5em; }
   .date ruby { background: #0078c9; color: #fff; }
   .date rt { background: #a0dae8; color: #000; padding: 2px; }
  </style>
 </head>
 <body>
  <p class="date">
   <ruby>
    23<rt>ноябрь</rt></ruby>
  </p>
  </body> 
</html>
